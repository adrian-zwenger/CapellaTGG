This package deals with the generation of consistent SystemComponent<->LogicalComponent tripels

it covers the generation of:
    * Components (SAB diagram)
    * Component containments (SAB diagram)
    * ComponentPkgs (SAB diagram)
    * ComponentPkg containments (context menu)

org.emoflon.ibex.tgg.rules.TransitionComponents.exchanges deals with generating exchange links between one or two components and their ports

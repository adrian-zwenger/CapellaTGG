This package deals with transitioning operational activities to SA.

ATTENTION:
    Activities can be transitioned as:
        a) Functions
        b) Capabilities
        c) Missions
    Only transition as Functions is covered here as it is the only transition offered by the Activity Explorer.


Diagrams interacting with these elements are:
    * OABD - Operational Activity Breakdown
    * ORB - Operational Role Breakdown
        + role definitions
        + interaction definitions
        + activity assignments to roles
    * OAIB - Operational Activity Interaction diagram
        + Activity definition (creation and naming)
        + Activity containment
        + Interaction definition
